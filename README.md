# Chef Portfolio team

## Endpoints

### All the endpoints start with https://chefportfolio10.herokuapp.com/


                                 
   # Register New User
   
   EndPoint :

                      ------------------------------------------------------
                                  Method      |       EndPoint 
                      ------------------------------------------------------
                                  POST        |     api/auth/register
                                  
                                  
                                  
   New User Object: 
 
                      -----------------------------------------------------
                                 username    |  Required and Unique
                      -----------------------------------------------------
                                 password    |  Required
                      -----------------------------------------------------
                                 email       |  Required and Unique
                      -----------------------------------------------------
                                 phone       |  Optional and Unique
                      -----------------------------------------------------
                                 state       |  Optional
                      -----------------------------------------------------
                                 city        |  Optional
                      -----------------------------------------------------
                                 address     |  Optional
                    
            

   # Login User
   
   EndPoint
   
   
                    -------------------------------------------------------------

                    -------------------------------------------------------------

   Login User Object: 
   
                    -----------------------------------------------------------

                    -----------------------------------------------------------
